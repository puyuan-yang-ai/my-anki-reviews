### 详细的分步操作指南 (Windows版)

请您跟着我的步骤，在您的 PowerShell 终端里操作。

#### 前提：确认您的环境
我已经看到您在 `(E:\conda_envs\chanlun_py311)` 环境中，这非常好！`quick_guide.md` 中明确指出**最低依赖版本为 python3.11**，您已经完美地满足了这个条件。

#### 第一步：进入项目根目录

您 `git clone` 之后，代码应该在 `chan.py` 文件夹里。您需要先进入这个文件夹。

```powershell
cd chan.py
```
*(请确保您是从 `E:\Python Code\pythonProject\LLM` 这个路径下执行此命令)*

#### 第二步：安装所有依赖包

现在您已经在项目根目录，并且激活了正确的 Conda 环境。接下来，我们使用 `pip` 来手动安装 `Script/requirements.txt` 文件中列出的所有依赖。

```powershell
pip install -r Script/requirements.txt
```

*   `pip install`: `pip`是 Python 的包安装工具。
*   `-r Script/requirements.txt`: `-r` 参数告诉 `pip` 读取一个文件，并安装里面列出的所有库。`Script/requirements.txt` 是依赖文件的路径。

这个命令会把所有必需的库都安装到您当前的 `chanlun_py311` 虚拟环境中，不会影响到您的 `base` 环境或其他项目。

#### 第三步：运行主程序

依赖安装完成后，您就可以启动项目了！根据 `quick_guide.md` 的指引，我们来运行 `main.py`。

在 Windows 的 Conda 环境中，我们通常直接使用 `python` 命令，而不是 `python3`。

```powershell
python main.py
```

执行这个命令后，程序就会开始运行，大概率会弹出一个带有K线图的窗口。
